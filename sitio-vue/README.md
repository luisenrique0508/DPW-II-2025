# sitio-vue

## 🚀 Configuración del Proyecto

### Instalación
```bash
npm install
```

### Desarrollo (servidor local)
```bash
npm run serve
```

### Producción
```bash
npm run build
```

### Linting
```bash
npm run lint
```
